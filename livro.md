# Meu livro

## **Carreira Dev: Hábitos para evoluir no desenvolvimento de software além do código**

### Sobre o livro

Neste livro, Jessica Nathany explora os aspectos fundamentais da carreira em desenvolvimento de software para você aprimorar suas habilidades interpessoais e expandir sua visão profissional. Uma vez fixados os pilares essenciais da jornada dev, a autora reúne hábitos para você crescer de forma estruturada, desde boas práticas de computação até soft-skills. 

Tudo recheado com aprendizados, experiências e depoimentos de pessoas da área. Você tem aqui um guia de carreira para desbravar o mercado de trabalho com prontidão para acompanhar suas tendências e se preparar para o futuro.
